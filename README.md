## Описание проекта
Один из моих первых проектов, который был сделан в июне 2017 года в виде проводной версии, а на текущий момент переведён на печатную плату.

Игровой процесс основан на воспроизведении случайно генерируемых последовательностей. Нарушение требуемых временных интервалов между нажатиями приводит к проигрышу.

В основе устройства используется микроконтроллер Arduino Nano, питание которого обеспечивается тремя батарейкам AA. Для световой индикаци
и применён RGB-светодиод с общим катодом, для звуковой — пьезоизлучатель.

## Список компонентов:
- Плата Arduino nano
- RGB DIP светодиод с общим катодом
- 3 резистора на 200 Ом
- Ультразвуковой пьезоизлучатель
- Кнопка
- Батарейный отсек формата АА
