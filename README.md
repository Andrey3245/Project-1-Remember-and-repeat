## Описание проекта
Один из моих первых проектов, который реализован в июне 2017 года. Игровой процесс основан на воспроизведении случайно генерируемых последовательностей. Нарушение требуемых временных интервалов между нажатиями приводит к проигрышу.

В основе устройства используется микроконтроллер Arduino Nano, питание которого обеспечивается тремя батареями формата AA. Для световой индикации применён RGB-светодиод с общим катодом, для звуковой — пьезоизлучатель.

## Список компонентов:
- Плата Arduino nano
- Батарейный отсек формата АА
- RGB DIP светодиод с общим катодом
- 3 резистора на 200 Ом
- Ультразвуковой пьезоизлучатель
- Выключатель
- Кнопка
- Батарейный отсек формата АА
